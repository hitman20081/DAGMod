# Selection Tokens

Tokens are one-time use items that allow you to make permanent character choices at selection altars.

### Race Tokens

Used at the **Race Selection Altar** to choose your racial heritage.

| Token | Race | Description |
|-------|------|-------------|
| Human Token | Human | Select the versatile Human race (+25% XP gain) |
| Dwarf Token | Dwarf | Select the mining-specialist Dwarf race |
| Elf Token | Elf | Select the swift and nature-attuned Elf race |
| Orc Token | Orc | Select the powerful warrior Orc race |

**Acquisition:**
- Automatically given when interacting with Race Selection Altar
- One token of each race provided
- Choose wisely - selection is permanent!

**Usage:**
1. Hold the token of your chosen race
2. Right-click the Race Selection Altar
3. Token is consumed and race is selected

---

### Class Tokens

Used at the **Class Selection Altar** to choose your combat class.

| Token | Class | Description |
|-------|-------|-------------|
| Warrior Token | Warrior | Select the melee tank Warrior class (5 abilities) |
| Mage Token | Mage | Select the magical spellcaster Mage class (4 abilities + wands + scrolls) |
| Rogue Token | Rogue | Select the agile assassin Rogue class (7 abilities via dual system) |

**Acquisition:**
- Automatically given when interacting with Class Selection Altar (after selecting race)
- One token of each class provided
- Choose based on your preferred playstyle

**Usage:**
1. Hold the token of your chosen class
2. Right-click the Class Selection Altar
3. Token is consumed and class is selected
